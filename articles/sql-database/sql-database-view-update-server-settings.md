# <a name="view-and-update-an-azure-sql-database-server-and-its-settings"></a>Visualizzare e aggiornare un server di database SQL di Azure le relative impostazioni

È possibile visualizzare e aggiornare un server di database SQL di Azure e le relative impostazioni usando il [portale di Azure]((sql-database-manage-portal.md), [PowerShell](sql-database-manage-powershell.md), l'API REST o C#. 

## <a name="view-and-update-an-azure-sql-server-settings-using-the-azure-portal"></a>Visualizzare e aggiornare le impostazioni di un server SQL di Azure tramite il portale di Azure
1. Aprire il pannello **SQL Server** nel [portale di Azure](https://portal.azure.com/). 

    ![nuova istanza di sql server](./media/sql-database-get-started/new-sql-server.png)

2. Fare clic sul server da usare e quindi scegliere l'impostazione desiderata nel pannello dei server SQL. 

    ![pannello di sql server](./media/sql-database-get-started/sql-server-blade.png)

> [!TIP]
> Per un'esercitazione introduttiva sull'uso del portale di Azure, vedere [Introduzione ai server di database SQL di Azure, ai database e alle regole del firewall usando il portale di Azure e SQL Server Management Studio](sql-database-get-started.md).
>
