# Gestione dati: Informazioni sulle tecnologie NoSQL in Azure

**Riepilogo:** Azure offre un'ampia gamma di opzioni per i carichi di lavoro NoSQL, ma può essere difficile stabilire le differenze tra tutte le opzioni disponibili. Questo documento illustra le tecnologie NoSQL in Azure e le differenze tra la gestione di queste tecnologie e i modelli relazionali per la gestione dei dati.

La tecnologia relazionale è da tempo l'approccio predominante per l'uso dei dati. Quando tuttavia si usano quantità elevate di tipi di dati diversi, come accade sempre più spesso attualmente, la gestione di questi dati presenta nuove difficoltà. La tecnologia relazionale è a volte, ma non sempre, la soluzione ottimale per la gestione di questi dati. In questa guida vengono illustrate le opzioni disponibili, con indicazione delle funzionalità offerte e dei vantaggi presentati da ognuna di esse.


**Sommario:**

* Dati in Azure: quadro generale	
* Tecnologie relazionali di Azure: Breve panoramica	
* Tecnologie NoSQL di Azure
	* Archivi di documenti: DocumentDB
	* Archivi di chiavi/valori: Tabelle
	* Archivi di famiglie di colonne: HBase
	* Analisi "Big Data": HDInsight
