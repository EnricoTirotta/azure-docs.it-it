# Gestione dati: Informazioni sulle tecnologie NoSQL in Azure

**Riepilogo:** Azure offre un'ampia gamma di opzioni per i carichi di lavoro NoSQL, ma può essere difficile stabilire quale sia quella più appropriata. In questo documento vengono illustrate le tecnologie NoSQL in Azure e viene spiegato come scegliere il servizio adatto per gestire i dati non relazionali, compresi i cosiddetti "big data", ovvero set di dati non strutturati o particolarmente vasti.

La tecnologia relazionale è data tempo l'approccio predominante per l'utilizzo dei dati. Tuttavia, in presenza di quantità molto elevate di dati di diverso tipo, la scalabilità su molti server risulta complessa. Inoltre, la tecnologia relazionale non è la soluzione ottimale per gestire alcuni tipi di dati, ad esempio grafici o documenti JSON. In questa guida vengono illustrate le opzioni disponibili, con indicazione delle funzionalità offerte e dei vantaggi presentati da ognuna di esse.
