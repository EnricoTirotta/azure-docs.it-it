# Report Tutta l'attività di accesso degli utenti di Azure Active Directory

*Questo documento fa parte della [guida alla creazione di report in Azure Active Directory](active-directory-reporting-guide.md).*

 >[AZURE.IMPORTANT]> - Il report Tutta l'attività di accesso degli utenti è un report Premium disponibile in Private Preview. Per accedere al report, inviare un messaggio di posta elettronica all'indirizzo aadreportinghelp@microsoft.com specificando il nome del proprio dominio (ad esempio **contoso.onmicrosoft.com**)


Il report Tutta l'attività di accesso degli utenti elenca tutti gli accessi alla directory eseguiti dagli utenti, indicando anche data e ora, UPN, indirizzo IP e l'applicazione a cui l'utente ha eseguito l'accesso.


 >[AZURE.IMPORTANT]> - Il report Tutta l'attività di accesso degli utenti è attualmente disponibile solo tramite l'API di creazione report. Per altre informazioni, vedere la pagina [Introduzione all'API di creazione report](active-directory-reporting-api-getting-started.md).
