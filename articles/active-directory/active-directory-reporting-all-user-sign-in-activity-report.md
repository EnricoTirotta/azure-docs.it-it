# Report Tutta l'attività di accesso degli utenti di Azure Active Directory

*Questo documento fa parte della [guida alla creazione di report in Azure Active Directory](active-directory-reporting-guide.md).*

 > [AZURE.IMPORTANT]
 >
 - Il report Tutta l'attività di accesso degli utenti è un report Premium disponibile in Private Preview. Per ottenere l'accesso al report, inviare all'indirizzo aadreportinghelp@microsoft.com un messaggio di posta elettronica con il proprio ID di tenant (GUID)


Il report Tutta l'attività di accesso degli utenti elenca tutti gli accessi alla directory eseguiti dagli utenti, indicando anche data e ora, UPN, indirizzo IP e l'applicazione a cui l'utente ha eseguito l'accesso.


 > [AZURE.IMPORTANT]
 >
 - Il report Tutta l'attività di accesso degli utenti è attualmente disponibile solo tramite l'API di creazione report. Per altre informazioni, vedere la pagina [Introduzione all'API di creazione report](active-directory-reporting-api-getting-started.md).
