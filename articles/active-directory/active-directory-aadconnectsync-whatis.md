# Servizio di sincronizzazione Azure AD Connect: Personalizzazione delle opzioni di sincronizzazione

Il Servizio di sincronizzazione Azure Active Directory Connect (Azure AD Connect Sync) è uno dei principali componenti di Azure AD Connect che si occupa di tutte le operazioni relative alla sincronizzazione dei dati di identità tra l'ambiente locale e Azure AD nel cloud. Dal punto di vista della sincronizzazione, il Servizio di sincronizzazione Azure AD Connect è il successore di DirSync, Azure AD Sync e Forefront Identity Manager con il connettore Active Directory configurato.
