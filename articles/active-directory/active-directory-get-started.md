# Introduzione all'uso di Azure AD

[Che cos'è Azure AD?](active-directory-whatis.md)<br> [Come funziona?](active-directory-works.md)<br> [Passaggi successivi](active-directory-next-steps.md)<br> [Altre informazioni](active-directory-learn-map.md)


Informazioni introduttive Azure AD è disponibile gratuitamente per chiunque. Per avere una directory, basta iscriversi a una sottoscrizione di valutazione di Azure. La directory gratuita resta di proprietà dell'utente, che si effettuino acquisti o meno. [Provare Azure AD adesso](http://azure.microsoft.com/trial/get-started-active-directory/)

## Come si fa a decidere quale directory usare per gli ambienti di test, staging e produzione?

Se è già stata eseguita la sottoscrizione di servizi cloud Microsoft, ad esempio Office 365, si ha già una directory in Azure AD ed è possibile usare quella. Quella directory autentica già gli utenti che accedono a Office 365. È quindi sufficiente iscriversi ad Azure usando lo stesso account aziendale o dell'istituto di istruzione che si usa per accedere a Office 365. A quel punto, sarà possibile gestire la directory nel portale di gestione di Azure, aggiungere directory per il test e così via, in base alle esigenze. [Altre informazioni su come gestire la directory per la sottoscrizione di Office 365 in Azure](active-directory-how-subscriptions-associated-directory.md#manage-the-directory-for-your-office-365-subscription-in-azure).

Se non si ha ancora una sottoscrizione per un servizio cloud Microsoft, verrà eseguito il provisioning della nuova directory per l'azienda o per l'istituto di istruzione al momento dell'iscrizione ad Azure e sarà possibile usarla come directory primaria.


## Come posso aggiungere il mio nome di dominio registrato?
Per aggiungere un nome di dominio personalizzato o usare il proprio nome di dominio registrato, vedere [Altre informazioni su come aggiungere un dominio personalizzato](active-directory-add-domain.md).

## Come posso aggiungere le informazioni personalizzate distintive dell'azienda?
Con Azure AD Premium, è possibile aggiungere il logo della società e altre informazioni personalizzate distintive dell'azienda alla pagina di accesso di Azure e alle pagine del pannello di accesso. [Aggiungere informazioni personalizzate distintive dell'azienda alla pagina di accesso e al pannello di accesso](active-directory-add-company-branding.md)

## Come posso integrare Azure AD con un'altra directory?
È possibile configurare la sincronizzazione della directory tra una directory locale e Azure AD, inclusa la sincronizzazione delle password, e abilitare l'accesso Single Sign-On alle applicazioni locali e nel cloud. [Altre informazioni sull'integrazione della directory](active-directory-aadconnect.md).

## Come posso creare e modificare gli utenti?
Creare un account per ogni utente che accederà a un servizio cloud Microsoft. Per impostazione predefinita, gli utenti non hanno autorizzazioni di amministratore, ma è possibile scegliere di assegnarle. [Altre informazioni sulla creazione di utenti di Azure AD](active-directory-create-users.md)

## Visualizzare i report di accesso e utilizzo
Azure AD fornisce numerosi report destinati agli amministratori. Con le edizioni di Azure AD a pagamento sono disponibili ulteriori report, oltre a un'API per la creazione di report per generare report personalizzati. [Altre informazioni sui report di Azure AD](active-directory-view-access-usage-reports.md)
