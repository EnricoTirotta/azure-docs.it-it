# Introduzione ad Azure AD Privileged Identity Management

## Avvio dell'estensione Privileged Identity Management

1. Creare una directory in Azure Active Directory.
2. Aggiungere gli utenti alla nuova directory, inclusi alcuni amministratori globali.

    >[AZURE.NOTE] Verificare di configurare gli utenti con gli account aziendali o dell'istituto di istruzione. Gli account Microsoft attualmente non possono iscriversi ad Azure AD PIM.

3. Accedere al [portale di Azure](https://portal.azure.com/) come amministratore globale di tale directory.
4. Fare clic sul nome utente nell'angolo superiore destro del portale di Azure.
5. Selezionare la directory appena creata.
6. Scegliere l’icona **Nuovo** nel riquadro di spostamento sinistro.
7. Selezionare **Sicurezza e identità** dal menu Crea.
8. Selezionare **Azure AD Privileged Identity Management**.
9. Lasciare **Aggiungi al dashboard** selezionato e quindi fare clic sul pulsante **Crea**. Verrà visualizzato il dashboard Privileged Identity Management
