# Altre informazioni su Azure AD

### Altri articoli su questo argomento
[Che cos'è Azure AD?](active-directory-whatis.md)<br> [Come funziona?](active-directory-works.md)<br> [Introduzione](active-directory-get-started.md)<br> [Passaggi successivi](active-directory-next-steps.md)<br> [Altre informazioni](active-directory-learn-map.md)

Si è già in possesso di una sottoscrizione di Azure. In che modo Azure AD è correlato? [Come vengono associate le sottoscrizioni a una directory](active-directory-how-subscriptions-associated-directory.md)

Per le attività amministrative di Azure AD quali la gestione degli utenti, la gestione del dominio o la configurazione dell'accesso Single Sign-On, usare i cmdlet del Modulo di Microsoft Azure Active Directory per Windows PowerShell. [Gestire Azure AD con Windows PowerShell](https://msdn.microsoft.com/library/azure/jj151815.aspx)

Usare l'API Graph di Azure Active Directory per l'accesso programmatico ad Azure AD tramite gli endpoint dell'API REST. [API Graph di Azure AD](https://msdn.microsoft.com/library/azure/hh974476.aspx)

È possibile aggiungere il proprio nome di dominio personalizzato ad Azure Active Directory per usarlo con i servizi online Microsoft. [Gestione del dominio Internet](active-directory-add-domain.md)

Per comprendere le limitazioni di utilizzo e altri limiti di servizio per il servizio di Azure Active Directory, vedere [Restrizioni e limiti del servizio Azure AD](active-directory-service-limits-restrictions.md)
