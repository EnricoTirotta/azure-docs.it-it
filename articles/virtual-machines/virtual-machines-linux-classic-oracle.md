# Software Oracle ospitato su macchine virtuali Linux in Azure

Configurare il primo o un successivo database Oracle di sviluppo/test o di produzione per l'esecuzione su una macchina virtuale Linux nei servizi infrastrutturali di Azure. Sfruttare la semplicità di configurazione e la possibilità di espandere rapidamente il carico di lavoro del database Oracle per includere la nuova capacità. È anche possibile installare software Oracle su [macchine virtuali Windows](virtual-machines-windows-classic-oracle.md).
