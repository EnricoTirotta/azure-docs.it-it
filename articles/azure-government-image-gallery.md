#  Raccolta di immagini di Microsoft Azure Government

<p> La raccolta di immagini di Microsoft Azure Government è un'esperienza simile al portale pubblico di Azure. I clienti hanno la possibilità di scegliere se distribuire immagini precedentemente create da Microsoft e i nostri partner oppure caricare i propri file VHD. Ciò fornisce la flessibilità di distribuire le proprie immagini standardizzate se necessario.

Nella tabella seguente viene riportato un elenco di immagini disponibili all'interno della raccolta di immagini di Azure Government. Alcune delle immagini già pronte includono licenze a pagamento in base al consumo per un software specifico. Consultare la pagina <a href="http://azure.microsoft.com/pricing/details/virtual-machines/">Prezzi macchine virtuali</a> per ottenere informazioni aggiuntive e collaborare con il rivenditore o il team degli account Microsoft per i prezzi specifici di Azure Government.
