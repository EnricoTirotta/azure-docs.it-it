# Ottenere la pre-approvazione tramite il programma di certificazione di Microsoft Azure

In questo articolo vengono fornite istruzioni su come diventare Microsoft Azure Certified. Per una panoramica dettagliata del programma di certificazione di Microsoft Azure e dei relativi vantaggi, visitare l'indirizzo [http://azure.com/certified](http://azure.com/certified)

> [AZURE.NOTE]La certificazione di Azure è disponibile solo se si intende creare una macchina virtuale, un modello di soluzione e servizi per gli sviluppatori per Azure Marketplace. Se non si sta creando uno di questi elementi, ignorare questo passaggio e procedere al **passaggio 1**, [Account venditore Microsoft: creazione e registrazione](marketplace-publishing-accounts-creation-registration.md)

## Introduzione
1. Il primo passaggio per la certificazione dell'applicazione o del servizio e la relativa integrazione in Azure Marketplace consiste nel compilare il modulo di richiesta nella pagina Web [Microsoft Azure Certified](https://azure.microsoft.com/marketplace/partner-program/).

2. Se si soddisfano i criteri di idoneità di base, verrà richiesto di condividere alcuni dettagli aggiuntivi sull'azienda e sull'applicazione o sul servizio che si desidera certificare. Una volta ricevute tutte le informazioni, l'applicazione verrà esaminata per garantire che sia adatta per Azure Marketplace. Se l'applicazione ottiene l'approvazione, avrà inizio la collaborazione con Microsoft per l'integrazione della soluzione in Azure Marketplace.

3. Se si lavora con un Partner Account Manager o un DX Partner Manager, richiedere loro di essere candidati per il programma di certificazione di Azure. In alternativa, visitare la pagina Web [Microsoft Azure Certified](http://azure.com/certified), fare clic su **"Richiedi informazioni"** e compilare il modulo di richiesta.

**Paesi di origine della vendita**

> [AZURE.WARNING]Per vendere i propri servizi in Microsoft Azure Marketplace, l'entità registrata deve trovarsi in uno dei paesi di origine della vendita approvati. Questa limitazione viene applicata per motivi legati ai proventi e alla tassazione. Questo elenco di paesi verrà ampliato nel prossimo futuro. Per un elenco completo, vedere la sezione 1b del documento disponibile all'indirizzo [http://go.microsoft.com/fwlink/?LinkID=526833](http://go.microsoft.com/fwlink/?LinkID=526833).

## Passaggio successivo
Dopo aver ricevuto la pre-approvazione dal programma di certificazione di Microsoft Azure, è possibile procedere al **passaggio 1**, [Account venditore Microsoft: creazione e registrazione](marketplace-publishing-accounts-creation-registration.md).
