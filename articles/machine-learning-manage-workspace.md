# Gestire un'area di lavoro di Machine Learning di Azure 
È possibile gestire le aree di lavoro di Machine Learning di Microsoft Azure (Azure ML) per gestire gli accessi, distribuire i servizi Web di produzione e gestire i servizi Web di Azure ML. Nella scheda Dashboard sono inoltre illustrati l'uso generale dell'area di lavoro e un riepilogo rapido delle informazioni sull'area di lavoro.  

Nella scheda **SERVIZI WEB** è possibile aggiungere, aggiornare o eliminare un servizio Web Azure ML.  

## Per visualizzare la pagina di gestione dell'area di lavoro ##
1.	Effettuare l'accesso con l'account Microsoft Azure.
2.	Nel pannello dei servizi di Microsoft Azure fare clic su **MACHINE LEARNING**.
3.	Nell'elenco **machine learning** fare clic su un'area di lavoro.
