## Azure IoT Suite
Azure IoT Suite è una soluzione aziendale che consente di iniziare rapidamente con una serie di soluzioni preconfigurate estensibili per scenari comuni IoT, ad esempio il *monitoraggio remoto* e i *dispositivi connessi*.

Le soluzioni preconfigurate sono:

- Sono soluzioni complete, funzionanti, end-to-end che includono dispositivi simulati per iniziare, servizi Azure preconfigurati come l’hub IoT Azure, gli hub eventi di Azure, l’analisi di flusso di Azure, Azure Machine Learning e Archiviazione di Azure e console di gestione specifiche per la soluzione.
- Contengono codice consolidato, pronto per la produzione che è possibile personalizzare ed estendere per implementare i propri scenari IoT specifici.

Si potrebbe inoltre essere interessati al servizio hub IoT Azure utilizzato da molte soluzioni preconfigurate. Il servizio hub IoT Azure fornisce molte delle funzionalità di "elaborazione dei dati e analisi" definite nell'architettura della soluzione di esempio.
