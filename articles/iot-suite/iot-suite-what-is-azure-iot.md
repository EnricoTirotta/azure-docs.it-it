## Azure IoT Suite

Azure IoT Suite di Microsoft è una soluzione aziendale che consente di iniziare rapidamente a usare un set di soluzioni preconfigurate estensibili per scenari IoT comuni, ad esempio il *monitoraggio remoto* e la *manutenzione predittiva*.

Le soluzioni preconfigurate:

- Sono soluzioni complete, funzionanti, end-to-end che includono dispositivi simulati per iniziare, servizi Azure preconfigurati come[ l’hub IoT Azure][], [gli hub eventi di Azure][], [l’analisi di flusso di Azure][], [Azure Machine Learning][], e [Archiviazione di Azure][], e console di gestione specifiche per la soluzione.
- Contengono codice consolidato, pronto per la produzione che è possibile personalizzare ed estendere per implementare i propri scenari IoT specifici.

Si potrebbe inoltre essere interessati al servizio [hub IoT di Azure][] usato da molte soluzioni preconfigurate. [Hub IoT Azure][] fornisce molte delle funzionalità di "elaborazione dei dati e analisi" definite nell'architettura della soluzione di esempio.
