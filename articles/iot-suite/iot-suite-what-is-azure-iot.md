## <a name="azure-iot-suite"></a>Azure IoT Suite
Microsoft Azure IoT Suite è una soluzione aziendale che consente di iniziare rapidamente a usare un set di soluzioni preconfigurate estendibili per scenari IoT comuni, ad esempio per il [monitoraggio remoto][lnk-preconfigured-solutions] e la [manutenzione predittiva][lnk-predictive-maintenance]. Tali soluzioni sono implementazioni dell'architettura di una soluzione IoT illustrata in questo articolo.

Quelle preconfigurate sono soluzioni end-to-end complete e funzionanti che includono dispositivi simulati per iniziare, servizi di Azure preconfigurati, come [Hub IoT di Azure][Hub IoT di Azure], [Hub eventi di Azure][Hub eventi di Azure], [Analisi di flusso di Azure][Analisi di flusso di Azure], [Azure Machine Learning][Azure Machine Learning] e [Archiviazione di Azure][Archiviazione di Azure], nonché console di gestione specifiche per la soluzione. Le soluzioni preconfigurate contengono codice consolidato, pronto per la produzione, che è possibile personalizzare ed estendere per implementare scenari IoT specifici.

Vedere anche il servizio [Hub IoT di Azure][Hub IoT di Azure], usato da molte soluzioni preconfigurate. [Hub IoT di Azure][Hub IoT di Azure] garantisce comunicazioni bidirezionali sicure e affidabili tra i dispositivi e il cloud usati nell'architettura delle soluzioni preconfigurate.
