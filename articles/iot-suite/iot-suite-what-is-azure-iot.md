## <a name="azure-iot-suite"></a>Azure IoT Suite
Microsoft Azure IoT Suite è una soluzione di livello aziendale che consente di iniziare rapidamente a usare un set di soluzioni preconfigurate estendibili per gestire scenari comuni di Internet delle cose, come il [monitoraggio remoto][lnk-preconfigured-solutions] e la [manutenzione predittiva][lnk-predictive-maintenance]. Tali soluzioni sono implementazioni dell'architettura di una soluzione IoT illustrata in questo articolo.

Si tratta di soluzioni preconfigurate complete e funzionanti di tipo end-to-end che includono:

- Dispositivi simulati per iniziare.
- Servizi di Azure preconfigurati come [Hub IoT Azure][Hub IoT Azure], [Hub eventi di Azure][Hub eventi di Azure], [Analisi di flusso di Azure][Analisi di flusso di Azure], [Azure Machine Learning][Azure Machine Learning] e [Archiviazione di Azure][Archiviazione di Azure].
- Console di gestione specifiche per la soluzione.

Le soluzioni preconfigurate contengono codice consolidato, pronto per la produzione, che è possibile personalizzare ed estendere per implementare scenari IoT specifici.

Vedere anche il servizio [Hub IoT Azure][Hub IoT Azure], usato da molte soluzioni preconfigurate. [Hub IoT Azure][Hub IoT Azure] garantisce comunicazioni bidirezionali sicure e affidabili tra i dispositivi e il cloud usati nell'architettura delle soluzioni preconfigurate.
