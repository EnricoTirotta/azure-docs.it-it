# <a name="what-products-can-be-published-via-the-cloud-partner-portal"></a>Quali prodotti possono essere pubblicati tramite il Portale per Cloud Partner

Il Portale per Partner Cloud è l'hub per pubblicare l'immagine della macchina virtuale di Azure Certified in Microsoft Azure Marketplace o l'applicazione Cortana Intelligence/Microsoft R Solutions in Microsoft AppSource.  

Le immagini della macchina virtuale sono immagini preconfigurate con installato un sistema operativo e una o più applicazioni. Per pubblicare l'immagine della macchina virtuale in Azure Marketplace, è necessario soddisfare i [prerequisiti tecnici](https://docs.microsoft.com/azure/marketplace-publishing/marketplace-publishing-vm-image-creation-prerequisites) per certificare l'immagine.
