# <a name="azure-government-marketplace-images"></a>Immagini di Marketplace di Azure per enti pubblici
## <a name="overview"></a>Panoramica
Marketplace di Azure per enti pubblici è un'esperienza simile al portale pubblico di Azure.  I clienti hanno la possibilità di scegliere se distribuire immagini precedentemente create da Microsoft e i nostri partner oppure caricare i propri file VHD.  Ciò fornisce la flessibilità di distribuire le proprie immagini standardizzate se necessario.

## <a name="variations"></a>Varianti
* Le immagini del Marketplace non sono attualmente supportate nelle versioni per il Dipartimento della difesa

Nella tabella seguente viene riportato un elenco di immagini disponibili all'interno di Marketplace di Azure per enti pubblici.  Alcune delle immagini già pronte includono licenze a pagamento in base al consumo per un software specifico.  Consultare la pagina <a href="http://azure.microsoft.com/pricing/details/virtual-machines/">Prezzi macchine virtuali</a> per ottenere informazioni aggiuntive e collaborare con il rivenditore o il team degli account Microsoft per i prezzi specifici di Azure Government.

### <a name="step-2"></a>Passaggio 2
Nel file Leggimi avviare il modello facendo clic sul pulsante o collegamento "Deploy to Azure Government" (Distribuisci in Azure per enti pubblici) nella parte superiore della pagina.

### <a name="step-3"></a>Passaggio 3
Una pagina del browser aprirà il pannello relativo alla distribuzione di modelli nel portale dedicato ad Azure per enti pubblici. Nel pannello dei parametri completare i campi obbligatori e fare clic su "OK".

### <a name="step-4"></a>Passaggio 4
Accettare i termini e le condizioni legali e inserire il segno di spunta nella casella "Aggiungi al dashboard", quindi fare clic su Crea.

> [!NOTE]
> È necessario un account valido di Azure per enti pubblici per continuare. Per consentire all'organizzazione di sfruttare i vantaggi di Azure per enti pubblici, iscriversi per una [versione di valutazione di Azure per enti pubblici](https://azuregov.microsoft.com/trial/azuregovtrial).
> 
> 

### <a name="step-5"></a>Passaggio 5
I pannelli verranno chiusi e si tornerà alla visualizzazione del dashboard del portale che mostra la distribuzione delle immagini. Questa operazione può richiedere alcuni minuti. Quando è in esecuzione, la soluzione della VM verrà visualizzata nel dashboard. Se non è stata selezionata l'opzione "Aggiungi al dashboard" nel passaggio 4, è necessario ricercare la VM nelle risorse.

> I modelli di soluzioni vengono forniti da Microsoft e da provider di terze parti solo come punto di partenza. Questi modelli vengono forniti "COME SONO" senza garanzie di alcun tipo, incluse le garanzie esplicite, implicite o derivanti dalla legge, e Microsoft non riconosce espressamente alcuna garanzia di commerciabilità, idoneità a uno scopo specifico o non violazione.
