# <a name="azure-government-marketplace"></a>Marketplace di Azure per enti pubblici
I partner interessati a pubblicare le offerte nel marketplace di Azure per enti pubblici possono trovare tutti i dettagli qui di seguito.

## <a name="publishing"></a>Pubblicazione
> [!NOTE]
> Se non si dispone della certificazione di partner del marketplace di Azure, prima di procedere eseguire le operazioni descritte [qui](../marketplace-publishing/marketplace-publishing-getting-started.md).
> 
> 

### <a name="step-1"></a>Passaggio 1
Accedere alla pagina [https://publish.windowsazure.com](https://publish.windowsazure.com).

### <a name="step-2"></a>Passaggio 2
Fare clic sull'offerta da pubblicare.

### <a name="step-3"></a>Passaggio 3
Fare clic su **SKUS** (SKU), quindi selezionare la casella Azure Government Cloud.

> [!NOTE]
> Sono supportate solo le SKU delle licenze Bring Your Own License (BYOL).  Questa opzione non è disponibile per le SKU con pagamento in base al consumo (Pay-as-You-Go, PayG).
> 
> 

![testo alternativo](./media/government-manage-marketplace-partner-1.png)

### <a name="step-4"></a>Passaggio 4
Fare clic sul collegamento + Add Certification (Aggiungi certificazione) per aggiungere collegamenti alle certificazioni offerte.

![testo alternativo](./media/government-manage-marketplace-partner-2.png)

### <a name="step-5"></a>Passaggio 5
Richiedere un account Microsoft Azure Government Cloud di prova per poter testare l'immagine nel portale di pubblicazione: [https://azuregov.microsoft.com/trial/azuregovtrial](https://azuregov.microsoft.com/trial/azuregovtrial).

Viene verificata l'idoneità come partner che opera per gli enti federali, statali o locali degli Stati Uniti, successivamente confermata tramite e-mail.  L'account di prova risulterà disponibile nell'arco di 3-5 giorni lavorativi.

### <a name="step-6"></a>Passaggio 6
Fare clic su Publish (Pubblica) e quindi su Push to Staging (Metti in gestione temporanea). 

![testo alternativo](./media/government-manage-marketplace-partner-3.png)

Verrà richiesto di immettere una sottoscrizione alla quale sia consentito accedere all'offerta in gestione temporanea. Immettere l'ID sottoscrizione dell'account di prova di Azure per enti pubblici appena acquisito.

![testo alternativo](./media/government-manage-marketplace-partner-4.png)

### <a name="step-7"></a>Passaggio 7
Una volta inserita l'offerta in gestione temporanea, è possibile testare l'immagine effettuando l'accesso al sito [https://portal.azure.us](https://portal.azure.us) con l'account di prova di Azure per enti pubblici.

### <a name="step-8"></a>Passaggio 8
Una volta convalidata l'immagine usando la sottoscrizione di prova, è possibile rendere disponibile l'offerta facendo clic su Publish (Pubblica) e richiedendo l'approvazione per passare alla produzione. 

![testo alternativo](./media/government-manage-marketplace-partner-5.png)

## <a name="next-steps"></a>Passaggi successivi
Per maggiori informazioni e aggiornamenti, iscriversi al [blog di Microsoft Azure per enti pubblici](https://blogs.msdn.microsoft.com/azuregov/).
