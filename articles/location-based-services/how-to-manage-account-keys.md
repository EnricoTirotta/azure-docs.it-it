# <a name="how-to-manage-your-azure-location-based-services-account-and-keys"></a>Come gestire l'account e le chiavi di Servizi Location Based di Azure

È possibile gestire l'account e le chiavi di Servizi Location Based di Azure tramite il portale di Azure. Dopo aver creato un account e una chiave di sottoscrizione, è possibile implementare le API nel proprio sito Web o nella propria applicazione per dispositivi mobili.

Se non si ha una sottoscrizione di Azure, creare un [account gratuito](https://azure.microsoft.com/free/?WT.mc_id=A261C142F) prima di iniziare.

## <a name="log-in-to-azure"></a>Accedere ad Azure 

Accedere al portale di Azure all'indirizzo http://portal.azure.com.

## <a name="create-a-new-account"></a>Creare un nuovo account

1. Fare clic sul pulsante **Nuovo** nell'angolo superiore sinistro del portale di Azure.

2. Cercare e selezionare **Servizi Location Based** e quindi fare clic su **Crea**.

3. Immettere le informazioni dell'account Servizi Location Based. 

![Immettere le informazioni dell'account nel portale](./media/how-to-manage-account-keys/new-account-portal.png)

## <a name="manage-keys-on-the-account-page"></a>Gestire le chiavi nella pagina dell'account

Nel menu a sinistra fare clic su **Chiavi** in Impostazioni.

![Gestire le chiavi dell'account nel portale](./media/how-to-manage-account-keys/account-keys-portal.png)

Usare le chiavi per eseguire l'autenticazione a Servizi Location Based di Azure e iniziare a sviluppare l'applicazione basata sulla posizione.

## <a name="delete-an-account"></a>Eliminare un account

È possibile eliminare un account facendo clic sul pulsante **Elimina** nel riquadro di spostamento superiore.

![Eliminare l'account nel portale](./media/how-to-manage-account-keys/account-delete-portal.png)
