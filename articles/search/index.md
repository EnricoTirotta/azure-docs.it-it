# <a name="azure-search-documentation"></a>Documentazione di Ricerca di Azure

Ricerca di Azure Ã¨ una soluzione di ricerca distribuita come servizio cloud che delega la gestione di server e infrastruttura a Microsoft, offrendo un servizio pronto per l'uso che consente di inserire dati e di aggiungere ricerche sulle applicazioni Web o mobili. Ricerca di Azure consente di implementare facilmente un'esperienza di ricerca affidabile nelle applicazioni usando una semplice API REST o .NET SDK senza dover gestire l'infrastruttura di ricerca o diventare esperti di ricerca.
