# <a name="azure-search-documentation"></a>Documentazione di Ricerca di Azure

Ricerca di Azure è un servizio di ricerca cloud gestito che contribuisce a creare un'esperienza di ricerca avanzata nelle app personalizzate e quindi di monitorare il servizio e le attività di indicizzazione e di query tramite le visualizzazioni del portale. Le funzionalità chiave includono la ricerca full-text scalabile in più lingue, la ricerca geospaziale, i filtri l'esplorazione in base a facet, query con completamento automatico, l'evidenziazione dei riscontri e gli analizzatori personalizzati.
