# Query in Ricerca di Azure
> [AZURE.SELECTOR]
- [Overview](search-query-overview.md)
- [Search Explorer](search-explorer.md)
- [Fiddler](search-fiddler.md)
- [.NET](search-query-dotnet.md)
- [REST](search-query-rest-api.md)

Le soluzioni personalizzate che integrano Ricerca di Azure per un'esperienza di ricerca incorporata devono includere il codice che crea una query di ricerca e la invia al motore di ricerca per l'elaborazione.

Per scrivere i metodi per l'esecuzione delle query, è possibile usare NET SDK o l'API REST. Per i test preliminari o l'esplorazione è possibile usare strumenti come Esplora ricerche, incorporato nel portale di Azure, per eseguire query su un indice oppure usare Fiddler per inviare qualsiasi richiesta valida al servizio di ricerca.

Visitare [Ricerca di Azure: Esercitazioni, demo video ed esempi](search-video-demo-tutorial-list.md) per un elenco di esempi e procedure dettagliate che illustrano come creare query nel codice. Quasi tutti gli esempi forniti includono codice che esegue query su un servizio di ricerca.
