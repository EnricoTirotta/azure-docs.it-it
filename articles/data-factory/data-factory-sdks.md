# SDK di Data factory di Azure

È possibile creare, monitorare e gestire le factory con il portale di Azure, Azure PowerShell, la libreria di classi .NET o con API REST.

Metodo | Percorso della risorsa | Riferimenti per gli sviluppatori
---------------------------------------------------- | ------------------------------ | -----------
Portale di Azure | [https://portal.azure.com/](https://portal.azure.com) | 
Azure PowerShell | Scaricare l'ultima versione di [Azure PowerShell](http://go.microsoft.com/?linkid=9811175&clcid=0x409) | [Informazioni di riferimento sui cmdlet](https://msdn.microsoft.com/library/dn820234.aspx)<p>[Informazioni di riferimento sugli script JSON](https://msdn.microsoft.com/library/azure/dn835050.aspx) </p> 
Libreria di classi .NET | .NET SDK di Data factory di Azure consente di creare, monitorare e gestire data factory di Azure e di estenderle con un'attività .NET. Per iniziare, vedere gli articoli [Usare attività personalizzate in una pipeline di Data factory di Azure](data-factory-use-custom-activities.md) e [Creare, monitorare e gestire le istanze di Data factory di Azure con .NET SDK per Data factory](data-factory-create-data-factories-programmatically.md).<p><b>Download della versione più recente di NuGet</b><br/>È possibile scaricare il pacchetto NuGet della libreria di gestione di Data factory di Azure dalla pagina: [https://www.nuget.org/packages/Microsoft.Azure.Management.DataFactories/](https://www.nuget.org/packages/Microsoft.Azure.Management.DataFactories/)</p><p>**Uso della console di Gestione pacchetti in Visual Studio**<br/>È possibile eseguire il comando seguente nella console di Gestione pacchetti di Visual Studio per scaricare la libreria di gestione di Data factory di Azure più recente</p>Install-Package Microsoft.Azure.Management.DataFactories</p> | [Informazioni di riferimento su .NET SDK](https://msdn.microsoft.com/library/dn883654.aspx)
API REST | È possibile usare l'API REST di Data factory per creare, monitorare e gestire data factory di Azure. | [Informazioni di riferimento sulle API REST](https://msdn.microsoft.com/library/dn906738.aspx)
