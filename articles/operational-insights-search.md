# Ricerca in Operational Insights
Un elemento fondamentale di Microsoft Azure Operational Insights è la funzionalità di ricerca, che permette di combinare e correlare qualsiasi dato di computer da più origini nell'ambiente. Gli Intelligence Packs sono basati anche sulla ricerca per offrire le metriche specifiche per una particolare area problematica.


## Cercare dati
È possibile usare la funzionalità di ricerca in Azure Operational Insights per trovare i dati desiderati. 

![Image of the Search tile](./media/operational-insights-search/overview-search.png)

Nella pagina **Search** è possibile creare una query e quindi, durante la ricerca, filtrare i risultati usando controlli facet. È anche possibile creare query avanzate per trasformare, filtrare e creare report sui risultati.

![Image of the Search page](./media/operational-insights-search/search-01.png)

Le query di ricerca più comuni sono visualizzate nella maggior parte delle pagine di ricerca. In tutta la console di Operational Insights è possibile fare clic su riquadri o analizzare altri elementi per visualizzarne i dettagli usando la funzionalità di ricerca.
È possibile usare facet e controlli di filtro per perfezionare i risultati. La digitazione assistita con il completamento automatico semplifica l'apprendimento della sintassi e permette di produrre rapidamente risultati.
