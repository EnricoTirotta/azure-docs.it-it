# Procedure dettagliate del Processo di analisi scientifica dei dati per i team

Le seguenti procedure dettagliate e complete illustrano i passaggi del Processo di analisi scientifica dei dati per i team sulla base di **scenari specifici** e spiegano come combinare strumenti cloud, strumenti locali e servizi in un flusso di lavoro o in una pipeline per creare un'applicazione intelligente.

## Uso di SQL Server
Nella procedura dettagliata [Processo di analisi scientifica dei dati per i team in azione: uso di SQL Sever](machine-learning-data-science-process-sql-walkthrough.md) viene illustrato come compilare e distribuire un modello di Machine Learning tramite SQL Server e un set di dati [Corse dei taxi di New York](http://www.andresmh.com/nyctaxitrips/) disponibile pubblicamente.


## Uso dei cluster Hadoop di HDInsight
Nella procedura dettagliata [Processo di analisi scientifica dei dati per i team in azione: uso dei cluster Hadoop di HDInsight](machine-learning-data-science-process-hive-walkthrough.md) si usa un [cluster Hadoop HDInsight](https://azure.microsoft.com/services/hdinsight/) per archiviare, esplorare e progettare funzionalità sui dati partendo da un set di dati [Corse dei taxi di New York](http://www.andresmh.com/nyctaxitrips/) disponibile pubblicamente.


## Processo di analisi scientifica dei dati per i team in azione: uso di cluster Hadoop di Azure HDInsight su un set di dati da 1 TB
Nella procedura dettagliata [Processo di analisi scientifica dei dati per i team in azione: uso di cluster Hadoop di Azure HDInsight su un set di dati da 1 TB](machine-learning-data-science-process-hive-criteo-walkthrough.md) viene presentato uno scenario completo che usa un [cluster Hadoop di HDInsight](https://azure.microsoft.com/services/hdinsight/) per archiviare, esplorare, progettare funzionalità e sottocampionare i dati da uno dei set di dati [Criteo](http://labs.criteo.com/downloads/download-terabyte-click-logs/) disponibili pubblicamente.

## Analisi scientifica dei dati con Spark in Azure HDInsight
Nella procedura dettagliata [Analisi scientifica dei dati con Spark in Azure HDInsight](machine-learning-data-science-spark-overview.md) si usa un Processo di analisi scientifica dei dati per i team in uno scenario end-to-end tramite il [cluster Spark di Azure HDInsight](https://azure.microsoft.com/services/hdinsight/) per archiviare, esplorare e progettare funzionalità sui dati da un set di dati [Corse dei taxi di New York](http://www.andresmh.com/nyctaxitrips/) disponibile pubblicamente.


## Analisi scientifica dei dati scalabile in Azure Data Lake
[Analisi scientifica dei dati scalabile in Azure Data Lake: procedura dettagliata end-to-end](machine-learning-data-science-process-data-lake-walkthrough.md) illustra come usare Azure Data Lake per eseguire attività di esplorazione dei dati e di classificazione binaria su un campione del set di dati relativo alle corse e alle tariffe dei taxi di NYC, in modo da prevedere se un passeggero pagherà la mancia.

## Procedura dettagliata di analisi scientifica dei dati end-to-end tramite i servizi R di SQL Server
[Procedura dettagliata di analisi scientifica dei dati end-to-end tramite i servizi R di SQL Server](https://msdn.microsoft.com/library/mt612857.aspx) usa un set di dati pubblico ben noto, ovvero il set di dati relativo alle corse dei taxi di New York. Usare una combinazione di codice R, dati di SQL Server e funzioni SQL personalizzate per compilare un modello di classificazione che indica la probabilità che il driver visualizzi un suggerimento su una particolare corsa di taxi. Distribuire quindi il modello R su SQL Server e usare i dati del server per generare punteggi in base al modello.

## Analisi avanzata nel database per gli sviluppatori SQL

L'obiettivo della procedura dettagliata [Analisi avanzata nel database per gli sviluppatori SQL](https://msdn.microsoft.com/library/mt683480.aspx) consiste nel fornire ai programmatori SQL istruzioni pratiche e funzionali sulla compilazione di una soluzione di analisi avanzata tramite i servizi R di SQL Server. Informazioni su come incorporare R in un'applicazione o in una soluzione di Business Intelligence eseguendo il wrapping del codice R in stored procedure.
