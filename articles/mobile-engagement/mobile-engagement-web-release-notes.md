#Note sulla versione di Azure Mobile Engagement SDK per Web

##2\.0.1 (10/06/2016)

-   SDK disabilitato in Internet Explorer 8 e Internet Explorer 9.
-   Rilevamento del browser Web Opera fisso.

##2\.0.0 (07/06/2016)

-   Versione iniziale di Azure Mobile Engagement
-   La configurazione di appId è stata sostituita con la configurazione della stringa di connessione.
-   Aggiunta di API per abilitare/disabilitare l'agente.
-   Sono stati introdotti miglioramenti per la sicurezza.
-   Uso di API JSON native dai browser.
-   L'API non riceve e non invia più messaggi tra i dispositivi.
-   Rimozione di callback correlati alla connessione XMPP deprecata. 
-   Questa versione non supporta la funzionalità Reach.
