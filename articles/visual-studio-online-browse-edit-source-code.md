# Sfogliare e modificare il codice sorgente

Nell'anteprima del portale di Microsoft Azure è possibile sfogliare e modificare facilmente il codice sorgente del progetto team di Visual Studio Online.


Dopo aver creato un progetto team nel portale e aver archiviato (Team Foundation Version Control) o eseguito il commit (GIT) del codice, è possibile accedere e sfogliare facilmente il codice dal portale.


1. Innanzitutto, aprire il pannello del progetto team e individuare le finestre di codice.  Quindi, nella parte di origine fare clic sul nome del repository da sfogliare (in questo caso, 'BrowseCode').  Nota: se sono presenti più di due repository è possibile fare clic sul numero del repository (1 in questo caso) per visualizzare l'elenco completo.
![Code Lens](./media/visual-studio-online-browse-edit-source-code/Code-Lens.png)
2. Con il pannello del repository aperto, è possibile fare clic sulla parte di codice per iniziare a sfogliare il codice sorgente.
![Repository Blade](./media/visual-studio-online-browse-edit-source-code/Repo-Blade.png)
3. Viene visualizzato un pannello di visualizzazione albero standard con l'intero progetto disponibile.  È possibile fare clic sui nodi per aprire la cartella o su uno specifico file (Index.cshtml in questo caso) per visualizzare o modificare il file.
![Tree Navigation](./media/visual-studio-online-browse-edit-source-code/Tree-Nav.png)
4. Per modificare un file, fare semplicemente clic sul pulsante Modifica nella barra dei comandi.  I contenuti del file passano in modalità di modifica ed è possibile apportare le modifiche desiderate. Suggerimento: Fare clic sul pulsante di ingrandimento in alto a destra per un'esperienza di modifica a schermo intero.
![Edit Mode](./media/visual-studio-online-browse-edit-source-code/Edit-Mode.png)
5. Al termine delle modifiche, fare clic sul pulsante Esegui commit (o Archivia) nella barra dei comandi.  Viene aperto un nuovo pannello in cui è possibile immettere il messaggio di commit.  Al termine fare clic su OK per eseguire il commit del file nel repository.
![Commit Code](./media/visual-studio-online-browse-edit-source-code/Commit-Code.png)
6. Nella parte superiore del pannello viene visualizzato un messaggio lampeggiante che indica la riuscita del commit nel repository.
![Commit Success](./media/visual-studio-online-browse-edit-source-code/Commit-Success.png)
