#Architettura
Risorse per la creazione di applicazioni che usano Microsoft Azure.

##Progetti di Microsoft Architecture

Microsoft sta pubblicando un set di progetti di architettura di alto livello che illustrano come creare tipi specifici di sistemi che usano i servizi di Microsoft Azure. Ogni progetto include un file 2D basato su Visio che è possibile scaricare e modificare, un file 3D più vivace in formato PDF per introdurre il progetto e un video che illustra il formato PDF in 3D. Vedere 
[Progetti di Microsoft Architecture](http://msdn.microsoft.com/dn630664).

![Microsoft Architecture Blueprint 3D diagram][BluePrint]

##Set di simboli e icone di Azure

[Scaricare il set di simboli e icone di Azure](http://www.microsoft.com/it-it/download/details.aspx?id=41937) per creare materiali tecnici che descrivono o usano Azure, ad esempio diagrammi di architettura, materiale didattico, presentazioni, fogli dati, infografiche e white paper. È possibile scaricare i simboli nei formati PPT, Visio o PNG. Il parere degli utenti è importante per Microsoft. Per questo motivo, il download contiene istruzioni per l'invio di commenti e suggerimenti.

I diagrammi 2D nei [Progetti di Microsoft Architecture](http://msdn.microsoft.com/dn630664) usano questi simboli. 

Nella versione successiva il set di simboli verrà espanso per includere i simboli di Cloud ed Enterprise. Spesso vengono rilasciati nuovi servizi di Azure prima dell'aggiornamento del download. Se si desidera visualizzare un'anteprima del nuovo set o dei simboli per i nuovi servizi di Azure, inviare un messaggio di posta elettronica all'indirizzo [AzureSymbols@microsoft.com](mailto:AzureSymbols@microsoft.com).  

Se si sono già usati i simboli, assicurarsi di compilare il breve [sondaggio](http://aka.ms/azuresymbolssurveyv1) di 5 domande per inviare commenti sulla loro utilità e sul loro uso.   

![Azure Symbol/Icon set][azure_symbols]
