# <a name="commands"></a>Comandi:
|||
|---|---|
|az mysql server|Gestisce i server. I server contengono database, data warehouse e pool elastici.|
|az mysql server create|Crea un server MySQL.|
|az mysql server delete|Elimina un server MySQL.|
|az mysql server list|Ottiene un elenco di tutti i server MySQL in un gruppo di risorse.|
|az mysql server show|Visualizza i dettagli di un server MySQL.|
|az mysql server update|Aggiorna un server.|
|az mysql server restore|Ripristina uno stato precedente di un server.|
|az mysql server firewall-rule|Gestisce le regole firewall di un server.|
|az mysql server firewall-rule create|Crea una regola del firewall.|
|az mysql server firewall-rule delete|Elimina una regola del firewall.|
|az mysql server firewall-rule list|Elenca le regole del firewall.|
|az mysql server firewall-rule show|Visualizza i dettagli di una regola del firewall.|
|az mysql server firewall-rule update|Aggiorna una regola del firewall.|
|az mysql db|Gestisce i database.|
|az mysql db list|Elenca tutti i database MySQL in un server.|
|az mysql server-logs list|Elenca i file di log per un server.|
|az mysql server-logs download|Scarica i log dal server in locale.|
|az mysql server configuration list|Elenca i parametri di configurazione del server.|
|az mysql server configuration show|Visualizza i dettagli dei parametri di configurazione server.|
|az mysql server configuration update|Modifica il valore di un parametro di configurazione server.|
|||
