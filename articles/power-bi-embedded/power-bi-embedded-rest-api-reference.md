# API REST di Power BI Embedded

Con **Microsoft Power BI Embedded** è possibile usare un'API REST per sviluppare a livello di codice applicazioni che integrano **report di Power BI** nelle proprie applicazioni.

Per informazioni sull'**API REST di Power BI Embedded**, vedere le [operazioni REST di Power BI](https://msdn.microsoft.com/library/mt711493).
