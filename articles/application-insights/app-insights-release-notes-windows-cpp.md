# Note sulla versione per Application Insights C++ SDK per Windows

[Application Insights SDK](app-insights-windows-cpp.md) invia dati di telemetria sull'app attiva ad [Application Insights](http://azure.microsoft.com/services/application-insights/), dove è possibile analizzarne l'utilizzo e le prestazioni.


#### Per installare l'SDK nell'applicazione

Vedere [Introduzione ad Application Insights per app di Windows Phone e Store](app-insights-windows-cpp.md).

#### Per eseguire l'aggiornamento all'SDK più recente 

* In Esplora soluzioni fare clic con il pulsante destro del mouse sul progetto e scegliere **Gestisci pacchetti NuGet**.
* Impostare il filtro in modo da visualizzare i pacchetti installati. 
* Selezionare i pacchetti di Application Insights installati e scegliere Azione: Aggiorna.
* Eseguire la [procedura di installazione](app-insights-windows-cpp.md) usando il nuovo numero di versione.
