# <a name="how-to-code-and-test-azure-functions-locally"></a>Come scrivere codice per le funzioni di Azure e testarle in locale
> [!NOTE]
> L'esperienza di sviluppo e uso di strumenti di Funzioni di Azure a livello locale è attualmente in fase di anteprima e verrà notevolmente migliorata prima della versione finale. Il modo più semplice per eseguire in locale l'host di Funzioni di Azure consiste nell'usare l'[interfaccia della riga di comando di Funzioni di Azure](https://go.microsoft.com/fwlink/?linkid=832752) e installarla da npm. Attualmente è supportato solo Windows. A breve sarà disponibile la documentazione completa. Per fornire commenti e suggerimenti o segnalare bug sull'interfaccia della riga di comando di Funzioni di Azure, presentare il problema nel repository GitHub [azure-webjobs-sdk-script](https://github.com/Azure/azure-webjobs-sdk-script).
