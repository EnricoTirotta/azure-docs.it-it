#### Errore durante il rilevamento dell'autenticazione

Durante il rilevamento di codice di autenticazione precedente, la procedura guidata ha rilevato un tipo di autenticazione non compatibile.

###### Elementi verificati

La procedura guidata tenta di rilevare le versioni del codice di autenticazione configurate con le versioni precedenti di Visual Studio. Se è stato visualizzato questo errore, è stato rilevato un tipo di autenticazione non compatibile. La procedura guidata rileva i tipi seguenti di autenticazione dalle versioni precedenti di Visual Studio:

-   Autenticazione di Windows
-   Account utente singoli
-   Account dell'organizzazione

Per altre informazioni, vedere [Scenari di autenticazione per Azure AD][Scenari di autenticazione per Azure AD].
