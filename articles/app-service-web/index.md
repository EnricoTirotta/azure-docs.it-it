<div class="content">
    <h1>Documentazione di App Web</h1>
    <p style="padding-bottom: 0px; max-width: 1050px;">App Web di Azure consente di creare e ospitare siti Web nel linguaggio di programmazione preferito, senza dovere gestire l'infrastruttura. Offre la scalabilità automatica e la disponibilità elevata, supporta sia Windows che Linux (anteprima) e consente le distribuzioni automatizzate da GitHub, Team Services o qualsiasi repository Git. Informazioni su come usare App Web di Azure con le guide rapide, le esercitazioni e gli esempi.</p>
