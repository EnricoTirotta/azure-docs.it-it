Per distribuire applicazioni in Azure, è necessario disporre di un account. Questo articolo illustra come creare un account di valutazione gratuito e come abilitare l'anteprima delle funzionalità di Azure, inclusi i Servizi BizTalk, HDInsight e Azure Backup, per l'account.

## <a name="create-an-azure-account"></a>Creare un account Azure
1. Aprire un Web browser e passare a <http://azure.microsoft.com>.
2. Se si dispone di una sottoscrizione MSDN, passare alla pagina [Benefici di Azure per gli abbonati MSDN](https://azure.microsoft.com/pricing/member-offers/msdn-benefits-details/) e seguire le istruzioni visualizzate per creare un account attivando i benefici per gli abbonati MSDN.
   
   In caso contrario, passare alla pagina relativa alla [creazione di un account di valutazione gratuito](https://azure.microsoft.com/pricing/free-trial/) e seguire le istruzioni visualizzate per creare un account di prova. Sarà necessario specificare un numero di carta di credito e un numero di telefono cellulare per dimostrare la propria identità, ma non verranno addebitati costi.

## <a name="enable-azure-preview-features"></a>Abilitare le funzionalità di anteprima di Azure
1. Passare a <https://account.windowsazure.com/> e accedere con l'account Azure.
2. Fare clic su **anteprima funzionalità** per visualizzare le anteprime disponibili.<br />
   ![aprire la scheda anteprima funzionalità][1]
3. Scorrere verso il basso fino alla funzionalità da abilitare e fare clic su **try it now** (prova adesso).<br />
   ![selezionare l'anteprima di una funzionalità][2]
4. Selezionare la sottoscrizione e fare clic sul segno di spunta.<br />
   ![selezionare la sottoscrizione][3]
