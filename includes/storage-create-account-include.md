## Creare un account di archiviazione di Azure

Il modo più semplice per creare il primo account di archiviazione di Azure consiste nell'usare il [portale di Azure](https://portal.azure.com). Per altre informazioni, vedere [Creare un account di archiviazione](../articles/storage/storage-create-storage-account.md#create-a-storage-account).

È possibile creare un account di archiviazione di Azure mediante [Azure PowerShell](../articles/storage/storage-powershell-guide-full.md), [l'interfaccia della riga di comando di Azure](../articles/storage/storage-azure-cli.md), [l'API REST di gestione](http://msdn.microsoft.com/library/azure/ee460799.aspx) versione 2014-10-01 o successive (distribuzioni classiche) o [l'API REST del provider di risorse di Archiviazione di Azure](https://msdn.microsoft.com/library/azure/mt163683.aspx) (distribuzioni ARM).
