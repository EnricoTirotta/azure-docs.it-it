In questa esercitazione vengono usati Android Studio e Android 4.2 o versione successiva. 

Questa esercitazione è basata sul progetto di guida introduttiva per Servizi mobili. Prima di iniziare questa esercitazione, è necessario completare l'esercitazione [Introduzione a Servizi mobili] o [Aggiunta di Servizi mobili a un'app esistente].

> [AZURE.NOTE] Il Mobile Services Android SDK è stato aggiornato alla versione 2.0. Se si sono completate le due esercitazioni precedenti in passato senza usare l'SDK aggiornato, è necessario ripeterle. Per verificare la versione dell'SDK usata dall'app, aprirla in Android Studio ed esaminare il contenuto della cartella **libs**. Il numero di versione dei file con estensione jar di Servizi mobili deve essere 2.0 o superiore.
