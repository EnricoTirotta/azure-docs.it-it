|L'identità del servizio gestito è attualmente una funzionalità in anteprima di Azure Active Directory offerta unicamente a scopo di valutazione. Prima di iniziare, assicurarsi di esaminare i [problemi noti](/azure/active-directory/msi-known-issues). Per altre informazioni sulle funzionalità in anteprima, vedere [Condizioni Supplementari per l'Utilizzo delle Anteprime di Microsoft Azure](https://azure.microsoft.com/support/legal/preview-supplemental-terms/).|
