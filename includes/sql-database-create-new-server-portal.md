## <a name="create-an-azure-sql-database-logical-server"></a>Creare un server logico del database SQL di Azure
Usare la procedura seguente nel portale di Azure per creare un server logico di database SQL di Azure.

1. Se non è ancora stato fatto, connettersi al [portale di Azure](http://portal.azure.com).
2. Fare clic su **Nuovo**, quindi su **Dati e archiviazione** e cercare **Database SQL**.
3. Fare clic su **Database SQL** per aprire il relativo pannello.
4. Specificare i valori per le proprietà del server seguenti:
   
   * Nome server
   * Accesso amministratore server
   * Password
   * Sottoscrizione, solo se sono disponibili più sottoscrizioni
   * Gruppo di risorse: nuovo o esistente
   * Percorso
5. Fare clic su **Create**. Nell'area di notifica si noterà che è stata avviata la distribuzione.
6. Attendere il completamento della distribuzione prima di procedere al passaggio successivo.
