### Prerequisiti
* Accesso a un database di SQL Server

Prima di poter usare Mashup SQL in un'app per la logica, è necessario autorizzare l'app per la logica a connettersi all'account Mashup SQL. Fortunatamente, è possibile eseguire questa operazione in modo facile dalla propria app per la logica nel Portale di Azure.

Ecco i passaggi per autorizzare l'app per la logica a connettersi all'account Mashup SQL:

1. Per creare una connessione a Mashup SQL, nella finestra di progettazione dell'app per la logica, selezionare **Mostra API gestite da Microsoft** nell'elenco a discesa, quindi immettere *Mashup SQL* nella casella di ricerca. Selezionare il trigger o l'azione che si desidera usare: ![Passaggio 1 Mashup SQL](./media/connectors-create-api-mashupsql/mashupsql-1.png)
2. Se non sono mai state create connessioni a Mashup SQL prima d'ora, verrà chiesto di fornire le credenziali di Mashup SQL. Queste credenziali verranno usate per autorizzare l'app per la logica a connettersi e ad accedere ai dati dell'account Mashup SQL: ![Passaggio 2 Mashup SQL](./media/connectors-create-api-mashupsql/mashupsql-2.png)
3. Si noti che la connessione è stata creata ed è ora possibile procedere con gli altri passaggi nell'app per la logica: ![Passaggio 3 Mashup SQL](./media/connectors-create-api-mashupsql/mashupsql-3.png)
