### Prerequisiti

- Un account Azure, che è possibile [creare gratuitamente](https://azure.microsoft.com/free)
- Un [database SQL di Azure](../articles/sql-database/sql-database-get-started.md) con relative informazioni di connessione, tra cui il nome del server, il nome del database, il nome utente e la password. Queste informazioni sono incluse nella stringa di connessione del database SQL: Server=tcp:*nomeserversql*.database.windows.net,1433;Initial Catalog=*nomedatabasesql*;Persist Security Info=False;User ID={your\_username};Password={password\_personale};MultipleActiveResultSets=False;Encrypt=True;TrustServerCertificate=False;Connection Timeout=30;

> [AZURE.NOTE] Quando si crea un database SQL di Azure, è inoltre possibile creare i database di esempio inclusi in SQL.

Ulteriori informazioni sui [database SQL di Azure](https://azure.microsoft.com/services/sql-database).
