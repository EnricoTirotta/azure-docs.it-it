Tornare alla finestra del browser che contiene il portale di Azure.

#### <a name="add-hostname"></a>Aggiungere il nome host

Fare clic sull'icona **+** accanto ad **Aggiungi il nome host**.

![Aggiungi il nome host](./media/app-service-web-tutorial-custom-domain/add-host-name-cname.png)

#### <a name="validate-hostname"></a>Convalidare il nome host

Digitare il nome di dominio completo per cui è stato configurato in precedenza il record CNAME, ad esempio `www.contoso.com`, quindi fare clic su **Convalida**.

Selezionare l'opzione **CNAME (www.example.com o qualsiasi sottodominio)** sotto l'intestazione **Tipo di record del nome host**.

Fare clic su **Aggiungi il nome host** per aggiungere il nome DNS all'app.

![Aggiungere il nome DNS all'app](./media/app-service-web-tutorial-custom-domain/validate-domain-name-cname.png)

La visualizzazione del nuovo nome host nella pagina **Domini personalizzati** potrebbe richiedere qualche minuto. Provare ad aggiornare il browser per visualizzare i dati più recenti.
