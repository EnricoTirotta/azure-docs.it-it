Questo set di dati è una versione leggermente modificata del set di dati disponibile nella raccolta StatLib della Carnegie Mellon University. Il set di dati è stato usato presso la American Statistical Association Exposition del 1983.<p> </p>I dati elencano il consumo di carburante per diverse automobili, in miglia per gallone, oltre a informazioni quali numero di cilindri, cilindrata, potenza, peso totale e accelerazione.<p> </p><b>Utilizzo:</b> prevedere il risparmio di carburante in base a 3 attributi discreti a più valori e 5 attributi continui. <p> </p><b>Ricerca correlata:</b> StatLib, Carnegie Mellon University, (1993).
